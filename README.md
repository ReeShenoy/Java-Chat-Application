# 💬 Java Chat Application

A real-time desktop chat application built using **Java Swing** and **Socket Programming**. This project demonstrates client-server communication through a simple and interactive graphical user interface.

## 🚀 Features

- Real-time messaging between client and server
- Java Swing-based GUI
- Client-Server architecture using sockets
- Clean and user-friendly interface
- Lightweight and easy to run

## 🛠️ Technologies Used

- Java
- Java Swing
- Socket Programming
- AWT
- Event Handling

## 📁 Project Structure

```
Java-Chat-Application/
│── Client.java
│── Server.java
│── icons/
│── README.md
```

## ▶️ How to Run

1. Clone the repository.
2. Open the project in your preferred Java IDE (VS Code, IntelliJ IDEA, Eclipse, etc.).
3. Compile all Java files.
4. Run `Server.java` first.
5. Run `Client.java`.
6. Start chatting!
